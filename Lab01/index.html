  1. COURSE SYNOPSIS

    Students begin this course by understanding how the internet functions in tandem with the various latest world wide web technologies. This course embarks on its main objective of exposing students to the numerous and diverse collection of current website design and application developement technologies. Students will learn the basic principles, protocols and currect practices that power the internet and the World Wide Web, how to design and implement a basic website, how to program scripts to serve the client's browser, how to program scripts that accesss information from the web server and how to integrate databases into the design of dynamic, data-driven websites

  2. COURSE LEARNING OUTCOMES

    The main learning outocomes of this course are:

    1. To illustrate the core concepts of web technologies, as well as its standards, developement framework, recent challenges and future trends
    2. To construct an interactive and responsive web application by using the learned client side scripting includes HTML5, CSS, Bootstrap 4, Javascript, JSON, AJAX or XML
    3. To develop a full-stack dynamic web developement in addressing the girn real-world business requirements by using MVC(Model-View and Controller)design pattern
